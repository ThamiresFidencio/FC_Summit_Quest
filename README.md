# 🏔️ FC Summit Quest

Um jogo 2D estilo plataforma desenvolvido com **Python + Pygame**, onde o objetivo é escalar uma montanha enfrentando obstáculos e inimigos até alcançar o topo.

---

## 🎮 Sobre o jogo

No *FC Summit Quest*, o jogador controla um personagem que precisa subir plataformas cada vez mais altas enquanto desvia de inimigos.

- Sistema de câmera vertical
- Obstáculos estratégicos
- Inimigos com movimentação
- Sistema de progresso (altura em metros)
- Condição de vitória ao alcançar o topo

---

## 🕹️ Controles

- ⬅️ ➡️ Setas → mover
- ⬆️ / Espaço → pular
- ESC → sair do jogo
